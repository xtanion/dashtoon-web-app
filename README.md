# dashtoon-web-app

## Installation (local)

```
cd dash-toon
npm install
npm run dev
```

## Usage
Inorder to generate 10 different comic panels, Start typing prompts separated by `,` (commas).

Adittionaly, you can download the image by clicking the `downlaod-icon` on the bottom-right corner.
